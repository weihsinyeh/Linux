## install vim

[Install vim Text Editor in Ubuntu](https://www.cyberciti.biz/faq/howto-install-vim-on-ubuntu-linux/)

command
```
sudo apt update
sudo apt upgrade
apt search vim
apt-cache search ^vim
apt-cache search vim | grep editor
apt show vim-tiny
apt show vim

sudo apt-get update 
sudo apt-get install vim
vim --mersion

```
